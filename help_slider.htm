• Create a slider object, and assign it an analog join equal to the configured base join + the decimal C-Bus group ID. eg. If you configured a base join of 1000, and you're wanting to control group 15, you'd use join 1015.
  • Assign the appropriate command to the release parameter of the slider (if you drag and drop the command onto the slider it will assign itself to the drag parameter, and this will cause excessive traffic on the C-Bus network.
  • Enable simulated feedback on the slider.
